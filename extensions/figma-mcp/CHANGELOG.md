# Changelog

All notable changes to the **figma-mcp** extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-17

### Added

- Figma design file tools via MCP server with SSE transport
- Dashboard panel for Figma file browsing
- Status widget showing Figma connection state
- Config schema with `figmaAccessToken` (vault-mapped)
