# hello-world/greet

Greet the user with a friendly message.

## When to use

Use this skill when the user wants to be greeted or wants a friendly hello message.

## Commands

### hello:greet
Greet the user with a friendly message.

**Arguments:**
- `--name <string>` — Name to greet (default: "World")

**Example:**
```
renre-kit hello:greet --name Alice
```
